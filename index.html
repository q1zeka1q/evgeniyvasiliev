MVP mõiste

Определение ключевой проблемы: Выявить основную проблему пользователей и определить минимальный набор функций, которые помогут её решить.

Что такое удалённый репозиторий?

Удалённый репозиторий – это место, где хранятся твои проекты в интернете (например, на сайте GitHub). Это нужно, чтобы можно было работать над проектом с разных компьютеров или с командой.

Как создать удалённый репозиторий?

Зайди на GitHub и войди в свой аккаунт.

Нажми кнопку "+" в правом верхнем углу и выбери "New repository"

Заполни поля

Нажми кнопку "Create repository". Всё, твой репозиторий готов!

Как подключить свой проект к удалённому репозиторию?

Открой в терминале (или командной строке).

Напиши команду, чтобы Git начал работать с проектом: git init

Подключи свой проект к удалённому репозиторию (ссылка будет на странице репозитория): git remote add origin https://github.com/твое-имя/мой-проект.git

Сохрани изменения: git add . , git commit -m "Моё первое сохранение"

Отправь проект на GitHub: git push

Если ты начинаешь новый проект

Клонируй репозиторий (скачай его к себе): git clone https://github.com/твое-имя/мой-проект.git

Зайди в папку с проектом: cd мой-проект

Добавь файлы и сохрани изменения: git add . , git commit -m "Добавил файлы" , git push
